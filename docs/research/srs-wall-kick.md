# SRS (Super Rotation System) 壁蹴りテーブル調査

## 概要

SRS (Super Rotation System) は The Tetris Company が定めた公式テトリスガイドライン(2001 年〜)で標準化された回転システム。本プロジェクトの SPEC.md §4.3 で「SRS を採用、壁蹴りオフセットは公式 SRS 準拠」と規定されている。

回転状態は 4 つ:

| 略号 | 角度 | 意味 |
|------|------|------|
| 0 | 0° | スポーン時の初期状態 |
| R | 90° (右) | 時計回りに 1 回転 |
| 2 | 180° | 半回転(上下/左右反転) |
| L | 270° (左) | 反時計回りに 1 回転(= 3 回時計回り) |

回転試行時は、まず基本回転を行い、衝突した場合は 5 つのオフセット(キックテスト 1〜5)を順に試す。すべて失敗すれば回転は不可。

## 座標系の注意

**SRS 標準テーブル**は `(x, y)` で表現され、`+x = 右`、**`+y = 上`** が慣例。

**本プロジェクト**は `(col, row)` で `+col = 右`、**`+row = 下`**。

したがって SRS テーブルから (drow, dcol) に変換する場合:

```
dcol = SRS_x
drow = -SRS_y    ← y 軸が反転
```

実装では「すでに行/列方向に変換済みのテーブル」を持つほうが事故が少ない。本ドキュメントでは下に **(drow, dcol) 形式**の変換済みテーブルも併記する。

## O ミノ

O ミノは回転しても 4 セル正方形のまま見た目が変わらないため、**回転を無視する**(あるいは状態だけ変更し位置オフセットなしで成功扱い)。SRS でも O ミノ用のキックテーブルは定義されていない。

## DOWN / UP ミノでの適用方針

本プロジェクトの SPEC §4.3 で「上向き重力のミノには SRS のオフセットを上下反転して適用」とある。

### Phase 1 (DOWN のみ)

上記 (drow, dcol) テーブルをそのまま使用すれば良い。

### Phase 2 で UP ミノに適用する場合(参考メモ)

UP ミノは「視覚的に正しい向きで現れて、その向きのまま中央へ進む」ため初期回転 180°(SPEC §4.2)で、移動方向は row 減少方向。

UP ミノで自然な壁蹴り感を保つには:

- 案 A: **オフセットの drow を符号反転**(`drow' = -drow`)し、回転状態の参照も上下対称に解釈する
- 案 B: 回転状態の意味を反転(0↔2, R↔L)して同じテーブルを引く

実装の単純さと SPEC §4.3 の文言「オフセットを上下反転」に最も忠実なのは案 A。確定は Phase 2 着手時に再調査する。
