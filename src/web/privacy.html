プライバシーポリシー

最終更新日: 2026-05-21

1. はじめに

本ポリシーは、Centerfall(以下「本サービス」)における利用者情報の取り扱いについて定めるものです。

2. 現在取得している情報

本サービスは現時点でサーバ側に利用者情報を一切送信しません。 取り扱う情報はすべてブラウザ内で完結します。

  • LocalStorage に保存される情報: キーバインド設定、モード別のハイスコア、その他のアプリ設定値。これらの情報は利用者の端末内にのみ保存され、提供者は閲覧できません。
  • Service Worker のキャッシュ: オフラインプレイのためにアプリ資材(HTML / CSS / JavaScript / 画像)をブラウザにキャッシュします。利用者の個人情報は含まれません。

3. Cookie の使用

本サービスは Cookie を使用しません。

4. アクセス解析・広告

本サービスはアクセス解析ツールおよび広告を使用していません。Google Analytics 等の第三者トラッキングも導入していません。

5. 第三者提供

本サービスは取得した情報を持たないため、第三者への提供は発生しません。

6. 取得情報の削除方法

ブラウザの設定から本サービスのサイトデータ(LocalStorage / Cache Storage)を削除することで、保存されたすべての情報を消去できます。

  • Chrome / Edge: 「サイトの設定」→ 該当ドメインの「データの消去」
  • Firefox: 「Cookie とサイトデータ」→ 該当ドメインを選択して削除
  • Safari: 「環境設定」→「プライバシー」→「Web サイトのデータを管理」

7. 将来の変更について

本サービスは今後、ハイスコアの永続化やランキング機能のために、クラウド上のデータベースへの情報送信を伴う機能を追加する可能性があります。 その際は、本ポリシーを更新したうえで、利用者からの同意を得る形式に変更します。

サーバ機能が追加された場合に取得が想定される情報の例:

  • アカウント識別子(ユーザー名等)
  • ハイスコア・プレイ統計
  • ログイン日時、IP アドレス等の技術情報(セキュリティのため)

これらは機能追加時に再度本ポリシーで明示します。

8. お問い合わせ

本ポリシーに関するお問い合わせは、GitHub リポジトリの Issues 経由でお願いいたします: solmen22/twin-tetris/issues

9. ポリシーの変更

提供者は、必要に応じて本ポリシーを変更することができます。変更がある場合は本ページの「最終更新日」を更新します。
