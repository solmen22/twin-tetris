    遊び方ガイド

    双方向重力テトリスの独自ルールを最短で理解するためのガイドです。

    1. このゲームについて

    双方向重力テトリスは、盤面の中央に境界線を挟んで上下から同時にミノが現れる新しいテトリスです。 上下の盤面が独立して振る舞いつつ、中央境界線で噛み合って大連鎖が発火します。

    UPPER FIELD
    ↓ DOWN ミノが落下
    中央境界線 (CENTER ROW)
    LOWER FIELD
    ↑ UP ミノが上昇

    2. 基本操作

    キーは 設定 / キーバインド 画面で全て変更可能です。以下はデフォルト:

    • ← → 左右移動
    • ↓ ソフトドロップ(+1pt / セル)
    • Space ハードドロップ(+2pt / セル、即着地)
    • ↑ X 右回転 / Z 左回転
    • C Shift ホールド
    • W 下方向を選択 / S 上方向を選択(USER CHOICE 時)
    • P Esc ポーズ / R リスタート

    3. DOWN ミノと UP ミノ

    盤面には 2 種類のミノが出現します:

    • DOWN ミノ: 上から落ちてくる。普通のテトリスと同じ感覚で扱える。↓ でソフトドロップは「下方向」に効く。
    • UP ミノ: 下から上に向かって「昇って」くる。↓ でソフトドロップは「上方向に加速」する。回転中心や接地判定の方向だけが反転し、操作の概念は同じ。

    4. 3 つのモード

    • RANDOM: 各ミノの方向(上/下)がランダムに決まる。一番カオスでスリルがある。
    • ALTERNATING: 上下が交互に出現。リズムよく積みたい人向け。
    • USER CHOICE: 各ミノの出現直前に W / S で方向を選択できる(0.5 秒の猶予あり)。最も戦略的。

    5. 中央境界線消去 — ×2 倍率 ⭐

    中央境界線(CENTER ROW)が完全に埋まると、その行が消えると同時に 上下の盤面が中央に向かって 1 段スライド します。 これがこのゲームの連鎖の起点です。中央が消えるとスライドによって新たに埋まった行が現れ、さらに消える…という連鎖が起こります。

    スコア倍率: 2 倍

    6. 上下同時崩落 — ×5 倍率 ⚡⚡

    1 回のロックで 中央 + 上の盤面 + 下の盤面 がすべて同時に消えると、上下同時崩落としてカウントされます。 本作で最も派手な得点源で、決まると爽快感がすごい。

    スコア倍率: 5 倍

    7. 連鎖ボーナス

    1 回のミノ確定から始まるカスケード(雪崩のような連続消去)が長いほど、終了時にチェインボーナスが加算されます。

    連鎖ボーナス: chain × 50pt

    8. スコアリングまとめ

    消去ライン数基本得点(× レベル)
    1 ライン100
    2 ライン300
    3 ライン500
    4 ライン (Tetris)800
    5 ライン以上800 + 200 × (超過分)

    これに 中央崩落 ×2 または 同時崩落 ×5 の倍率と、連鎖ボーナスが乗ります。

    9. 上達のコツ

    • 序盤は上下それぞれを別の盤面として処理して、中央を「揃いそうな状態」に整えていく。
    • 中央境界線を意識して凹凸を作りすぎないように積む。
    • USER CHOICE モードでは、Hold を活用して欲しい方向のミノを引っ張り出せる。
    • 同時崩落 ×5 はゲーム全体の得点を一気に押し上げる。狙えるなら積極的に。
